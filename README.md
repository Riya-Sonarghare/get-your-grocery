🛒 GetYourGrocery
GetYourGrocery is a multi-role grocery web application developed using Spring Boot that connects customers and sellers on a single platform. Customers can explore and purchase grocery products, while sellers can manage products and orders through a dedicated dashboard.

✨ Features
🔐 Secure Login & Registration
👤 Customer and Seller roles
🛍 Browse grocery products
❤️ Save favorite products
📦 Place and manage orders
➕ Add & manage products
📊 Seller dashboard
⚡ Form validation and exception handling

🛠 Tech Stack
Backend: Java, Spring Boot, Spring Data JPA, Spring MVC
Frontend: Thymeleaf, HTML, CSS, Bootstrap
Database: Oracle Database
Tools: Maven, Lombok

🚀 Getting Started
Clone the repository:

git clone https://github.com/Riya-Sonarghare/GetYourGrocery.git
Configure database credentials in:

application.properties
Run the application:

mvn spring-boot:run
Open in browser:

http://localhost:8080
